ダウンロードフォルダ振り分け

特定のサイトやファイルタイプからのダウンロードを指定したサブフォルダに、優先度付きルールで自動的に振り分けます。

機能概要

Webからのダウンロード完了時にファイルを整理する拡張機能です。事前に設定したルールに従い、ファイルを指定フォルダへ自動的に移動します。

  • 柔軟なルール設定: URLパターン、キーワード、正規表現を使用可能
  • ファイルタイプ別: 拡張子ごとに保存先を指定可能
  • 優先度設定: 競合するルールを優先順位で制御
  • 手動実行: 右クリックメニューからの即座振り分け

使い方

  1. 拡張機能アイコンをクリックし、設定画面を開きます。
  2. 「新しいルールを追加」ボタンをクリックします。
  3. 条件（URLやファイルタイプ）と保存先フォルダを入力し、保存します。
  4. 次回から条件に一致するファイルが指定フォルダへ自動保存されます。
