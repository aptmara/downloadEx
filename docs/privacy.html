最終更新: 2026年2月9日

このページでは、「ダウンロードフォルダ振り分け」（以下「本拡張機能」）におけるユーザー情報およびデータ処理に関する方針を説明します。

1. データ収集の有無

本拡張機能は、ユーザーの個人情報、ブラウザ履歴、その他の機密データを外部サーバーへ送信しません。すべての処理はユーザーのブラウザ（ローカル環境）でのみ実行されます。

ルール設定や履歴などのデータは、ブラウザ内の `chrome.storage.local` API を使用してローカルに保存されます。

2. 権限の使用理由

本拡張機能が要求する権限と、その使用目的は以下の通りです。

  • downloads: ファイルのダウンロード開始を検知し、設定されたルールに基づいて保存先の決定やファイル名の変更を行うために使用します。
  • storage: 作成された振り分けルールや設定内容をブラウザ内に保存し、次回起動時にも適用できるようにするために使用します。
  • contextMenus: ブラウザ上のリンクなどを右クリックした際、そのファイルを指定ルールで即座にダウンロード・振り分けするためのメニューを表示するために使用します。

3. 第三者への提供

収集するデータが存在しないため、第三者へのデータの提供や共有は一切行いません。

4. 免責事項

本拡張機能の使用により生じたいかなる損害についても、開発者は責任を負いません。本拡張機能は「現状有姿」で提供されます。

5. 連絡先

プライバシーに関するご質問やご報告は、GitHubのリポジトリへお願いいたします。
